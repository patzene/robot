# Ρομπότ Line Follower - Ομάδα 9

Έργο για το μάθημα "Ενσωματωμένα Συστήματα" (Εαρινό Εξάμηνο 2026) - Τμήμα Πληροφορικής ΔΠΘ.

## Περιγραφή
Αυτόνομο όχημα τύπου line follower βασισμένο στον μικροελεγκτή Maker Pi RP2040, το οποίο χρησιμοποιεί αναλογικούς και ψηφιακούς αισθητήρες υπερύθρων και αλγόριθμο PD (Proportional-Derivative) για τον έλεγχο της πορείας του.
